## About The Project

You can explore predefined categories such as politics, technology, sports and more. Additionally, for those with specific interests, the app allows you to create and search for your own custom categories, ensuring that you have access to the stories that matter most to you.

## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Get a free API Key at [https://newsapi.org/](https://newsapi.org/)
2. Clone the repo
   ```sh
   git clone https://github.com/SaschaStg/news-today.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Create a `.env.local`  file in your root directory
   ```js
   NEWS_API_KEY=<Your API key>
   ```
